Setting up your machine.

  1. Install racket.
    Racket lets you write Lisp code, and comes with an ide that can be called in the termianl using "drracket"
  2. Run "raco pkg install simply-scheme".
    This allows you to use the pre-defined functions we will need.
  3. Make sure at the bottom of Dr. Racket it says "determine language from source". If it does not, click "Language" on the top bar, and select "The Racket Language".
