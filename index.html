Install racket. This comes with an ide called drracket. Run it via "drracket"

Then run "raco pkg install simply-scheme". This will give you access to the version of Scheme the proff. uses. Make sure at the bottom of Dr. Racket it says "determine language from source". If it does not, click "Language" on the top bar, and select "The Racket Language".
